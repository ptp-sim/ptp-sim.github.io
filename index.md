
# Simulation of PTP (IEEE 1588)

**ptp-sim** is a community effort to provide a simulation for the Precision Time Protocol (PTP) as specified in IEEE 1588.
It is based on OMNeT++ and INET.

## Project overview

* OMNeT_Utils
* libPTP
* PTP_Simulations
* libPLN
